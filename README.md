# 🌐 Sirius Core

Portal de tecnologia com artigos sobre redes, desenvolvimento, segurança e IA.

Acesse: [siriuscore.com.br](https://siriuscore.com.br)

## Tecnologias

- HTML5
- CSS3 (dark mode, responsivo, sem frameworks)
- JavaScript vanilla

## Páginas

- **Home** — hero, artigos em destaque e seção sobre o autor
- **Artigos** — listagem com filtros por categoria
- **Artigo** — template de post com sidebar e artigos relacionados
- **Sobre** — portfólio do autor com skills, timeline e contatos

## Rodar localmente

Basta abrir o `index.html` no navegador ou usar a extensão Live Server no VSCode.
